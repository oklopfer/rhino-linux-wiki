Welcome to the Rhino Linux Wiki, where you can find everything you'd want to know about Rhino Linux!

## What is Rhino Linux?

Rhino Linux is an Ubuntu based, rolling release Linux distribution with Pacstall and XFCE at our core. A rolling release distribution is a Linux distribution which recieves continuous package updates, and as such there is no major update. A rolling release model offers new and experienced users a new way to utilize their desktop PC, without the hassle of major upgrades

Check out our github for more info: https://github.com/rhino-linux
